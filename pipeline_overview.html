7. 모델별 구체적 피처 — "어떤 컬럼을 왜 쓰는가"

7-1. Track A 공통 피처 (6개 모델: ElasticNet, RF, XGB, LGBM, MLP, TabNet)

모두 동일한 (drug, cell_line) 쌍별 피처 매트릭스 X를 입력받음. features.py build_feature_matrix()

그룹컬럼 예시개수타입값 범위왜 필요한가
타겟 유전자 이진BRCA1=1, TP53=0, EGFR=1, ERBB2=1, ...~2,000int 0/1{0, 1}약물이 타겟하는 유전자 = 감수성 예측의 핵심. L1(ElasticNet)이 자동 선택, RF가 중요도 산출
pChEMBL meanpchembl_mean1float5.0~10.0약물-타겟 결합 강도. 높을수록 potent → 감수성 확률 높음
ADMET gatesgate1_druglikeness, gate2_solubility, gate3_permeability, gate4_toxicity, all_pass5int 0/1{0, 1}약물이 인체에 도달 가능한지. ADMET 실패 약물은 아무리 IC50 좋아도 무의미
Pathway overlappw_overlap_count, pw_overlap_ratio2int/float0~7600 / 0~1타겟이 속한 생물학적 경로 수. 다경로 약물이 단일경로보다 효과적 (Subramanian 2005)
Target expressiontarget_expr_mean, target_expr_std, target_expr_max3float0~15 (log2)타겟 유전자가 암에서 실제 발현되는지. 미발현 타겟은 약물이 작용할 수 없음
CRISPR dependencydep_BRCA1, dep_TP53, dep_MYC, ... (top-50)50float-2.0~0.5 (Chronos)세포주별 필수 유전자가 다름. 동일 약물도 세포주마다 반응 다름 (DepMap)
PRISM sensitivityprism_mean_auc, prism_std_auc, prism_n_drugs3float0.3~1.0 / 0~100세포주의 전반적 약물 감수성 프로필. 민감 세포주는 대부분의 약물에 민감

합계: ~2,064 피처 × 7,200 샘플. Label: GDSC binary (0=resistant, 1=sensitive)

모델피처 처리 방식왜 이 모델?근거

9. 앙상블 (RRF 7-ranking)

src/ml/ensemble.py — RRF(d) = Σ 1/(k + rank(d)), k=60
근거: Cormack et al. 2009 SIGIR — 스케일 무관 순위 통합

Track A: best model OOF predictions → drug별 평균  ──┐
Track B: AE latent → target gene norm → drug 점수   ──┤
Track B: VAE latent → target gene norm → drug 점수  ──┤
Track B: GNN embedding → cosine similarity           ──┼──▶ RRF ──▶ Final Ranking
Track C: Network Proximity z-score                    ──┤
Track C: Signature Reversal score                     ──┤
Track C: Pathway Overlap score                        ──┘

후처리: 피처 단계 ADMET 컬럼 + Open Targets association score 부착. 최종 임상 진입 ADMET 4 Gate는 §2 Bedrock 이전 별도 박스에서 수행.
신뢰도: 2+ Track에서 top-20 → "high", 1 Track → "medium"

ADMET을 “두 번” 돌린다는 뜻이 아님. (1) 피처 단계: ADMET 예측값을 테이블 컬럼으로 넣어 모델이 학습·랭킹에 활용. (2) 최종 게이트: 숏리스트에 대해 임상·개발 관점에서 통과/탈락을 정하는 정책 단계. 동일 예측을 앞에서는 입력 신호로, 뒤에서는 컷오프로 재사용할 수 있어 계산은 한 번·역할은 두 층일 수 있음. Gate 전용 도구를 따로 두면 계산 파이프가 둘로 갈라질 수 있음.
